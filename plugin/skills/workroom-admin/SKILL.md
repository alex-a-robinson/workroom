# Workroom team overview

**Trigger on:**
- User asks to see team progress or admin view
- User asks to invite a teammate
- User wants to check team adoption or activity
- User asks about their team's Workroom status

---

## What this skill does

Opens your **team management view** — a high-level dashboard for seeing adoption, progress, and engagement across your team.

The artefact shows:
- **KPI tiles** — team size, onboarding rate, routines completed, time-to-first-win
- **Invite card** — generate and copy an invite code to add teammates
- **Team progress table** — each person's role, completion %, last activity, routines done
- **Recent activity feed** — timeline of team wins and engagement

---

## Current state: Preview

This is a **placeholder version** with honest data. You're the only team member shown, and the invite system is a mock. When the real invite distribution ships, this view will:
- Sync with your actual invite system
- Show real team member status
- Update as teammates onboard and complete routines
- Track team-wide metrics automatically

---

## Using the invite code

Right now, the invite code is a preview. When real invites ship:
1. Copy the code from the artefact
2. Share it with a teammate
3. They click the code to install the plugin and join your team view
4. You see their progress here

---

## Why this matters

Without this view, you can't tell if Workroom is actually working for your team. With it, you can:
- Spot who's engaged and who's stuck
- See which routines your team loves (and which they skip)
- Measure time-to-first-win and ROI
- Decide whether to expand Workroom across your org
