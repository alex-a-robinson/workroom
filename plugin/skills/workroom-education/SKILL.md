# Your Workroom learning path

**Trigger on:**
- User requests to see their learning path, education, or progress
- User asks "what can I learn", "what's next", "show me my progress"
- User asks about routines or available modules

---

## What this skill does

Opens your **personal learning path** — a curated sequence of role-specific routines tailored to your work and connected tools.

The artefact shows:
- **Your progress** — how many routines you've completed
- **In progress** — the routine you're currently working on (if any)
- **Available next** — 3 routines you can start right now
- **What's coming** — a preview of upcoming modules

Each routine is a 5–15 minute hands-on module that teaches you a new capability. Routines are locked until you connect the tools they require.

---

## How it works

1. The skill reads your progress file (created during onboarding).
2. If no progress file exists, it shows a friendly empty state inviting you to run onboarding.
3. If preview mode (no MCP access), it loads a sample learning path.

---

## Empty state

If you haven't completed onboarding yet, the artefact shows an empty state with a button to **"Run onboarding"**. Once you onboard, your learning path appears automatically.

## Tips

- **Start with available routines** — these don't require any extra tool connections.
- **Routines lock when you need a tool** — connect that tool to unlock.
- **You control the pace** — learn one routine a week, or tackle three in a day. No pressure.
- **Save time-consuming routines for focus time** — most are 5 min, but some deserve a quiet hour.

---

## Next steps

- **Complete a routine** — each one makes you a little faster at your job.
- **Save workflows that stick** — turn a good manual process into a scheduled task.
- **Tell me what you need** — if a routine doesn't quite fit your workflow, ask me to adapt it.
