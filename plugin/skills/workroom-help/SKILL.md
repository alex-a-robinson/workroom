# Welcome to Workroom

Workroom is an activation plugin for Claude that ships onboarding, knowledge browsing, the Pip framework for team teaching, and a starter workspace scaffold.

## What you can do right now

- **workroom-onboarding** — Spawn the onboarding artefact and scaffold a Workroom-style workspace into your chosen directory.
- **workroom-file-viewer** — Open an Obsidian-style markdown knowledge-base browser.
- **workroom-pip** — Load the Pip coaching framework as in-session behaviour.
- **workroom-info** — Check the plugin version, what's loaded, and recent changelog.
- **workroom-help** — Show this guide (you're reading it now).

## Next step

Run the **workroom-onboarding** skill to create your first workspace. It will set up a project scaffold with templates, conventions, and learning paths.

**Note:** This is Workroom v0.3.x, the first real release. Hooks are placeholder stubs; no custom Workroom MCP is shipped yet. See the changelog for what's planned.
